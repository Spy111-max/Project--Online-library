# Project--Online-library

A lightweight online library web app for youngsters to browse and download books.

## Features
- Browse books across categories like biography, autobiography, novel, and self-help.
- Filter and sort books by **genre**, **author**, and **season**.
- Download books as PDF files.
- Every downloaded book is saved in a **virtual library shelf** (stored in browser local storage).

## Run
Open `index.html` in any modern browser.
